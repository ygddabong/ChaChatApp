# ChaChatApp
What's app clone app

## Getting Started
You must install pod file

#### Snap Kit
<code>
  pod 'SnapKit'
</code>

#### Firebase
<code>
  pod 'Firebase/Database'
</code>
<code>
  pod 'Firebase/Auth'
</code>
<code>
  pod 'Firebase/Storage'
</code>  
<code>  
  pod 'FirebaseUI'
</code>

#### Kingfisher
<code>
  pod 'Kingfisher'
</code>

#### MessageKit
<code>
  pod 'MessageKit'
</code>
